# Demo

Add the following GIF files to this directory:

1. **expand-collapse.gif** — Shows the player in expanded portrait mode, then user taps to collapse it into a mini-player in the corner.

2. **drag-snap.gif** — Shows the mini-player being dragged around the screen, snapping to corners with spring physics animation.

3. **landscape.gif** — Shows the player rotating to landscape mode with fullscreen video, then dragging down to exit.
